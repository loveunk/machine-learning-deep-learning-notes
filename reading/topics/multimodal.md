# 多模态阅读索引

这个主题关注视觉语言模型、文档理解、视频理解、多模态 Agent 和真实业务中的多模态数据流。

## 当前关注方向

- VLM 基础能力：图像理解、OCR、视觉问答。

- 多模态 Agent：视觉观察、工具操作、浏览器或桌面环境。
- 评估：视觉 grounding、引用、定位、误识别。
