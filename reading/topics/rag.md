# RAG 阅读索引

这个主题关注检索增强生成、向量检索、rerank、长上下文、GraphRAG、评估和引用可信度。

## 当前关注方向

- 检索质量：chunking、embedding、metadata、hybrid search、rerank。
- 生成可信度：引用、忠实度、拒答、答案可追溯。
- 评估：检索召回率、答案正确率、faithfulness、人工标注集。
- RAG 与 Agent：何时把检索作为工具，何时保持固定问答链路。
